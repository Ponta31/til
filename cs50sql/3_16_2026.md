## SELECT, LIMIT, WHERE

> SELECT "title" FROM "longlist";

--> Double quote shows column

--> Single quote shows strings


We can limit output
> SELECT "title"  FROM "longlist"  LIMIT 10;


We need to select rows based on a condition
> SELECT "title", "author" FROM "longlist" WHERE "year" = 2023;


We can use operators:

```
SELECT "title", "format" 
FROM "longlist" 
WHERE "format" != 'hardcover';
```
It's the same as <>
```
SELECT "title", "format" 
FROM "longlist" 
WHERE "format" <> 'hardcover';
```
It's also the same using 'NOT'
```
SELECT "title", "format" 
FROM "longlist" 
WHERE NOT "format" = 'hardcover';
```

- We can also use 'AND', 'OR'
```
SELECT "title", "author" 
FROM "longlist" 
WHERE "year" = 2022 OR "year" = 2023;
```

- We can add more
```
SELECT "title", "format" 
FROM "longlist" 
WHERE ("year" = 2022 OR "year" = 2023) AND "format" != 'hardcover';
```
