Define a function that takes an integer argument and returns a logical value true or false depending on if the integer is a prime.

Per Wikipedia, a prime number ( or a prime ) is a natural number greater than 1 that has no positive divisors other than 1 and itself.

Requirements
You can assume you will be given an integer input.
You can not assume that the integer will be only positive. You may be given negative numbers as well ( or 0 ).
NOTE on performance: There are no fancy optimizations required, but still the most trivial solutions might time out. Numbers go up to 2^31 ( or similar, depending on language ). Looping all the way up to n, or n/2, will be too slow.


```
def is_prime(num):
    if num < 2:
        return False
    for i in range(2, math.isqrt(num)+1):
        if num % i == 0:
            return False
    return True

```

Why √n?
If n has a factor larger than √n, it must also have a factor smaller than √n. So you never need to check beyond √n.
例えば：n = 36 の場合、因数のペアを考えると：
2×18,3×12,4×9,6×62 \times 18, \quad 3 \times 12, \quad 4 \times 9, \quad 6 \times 62×18,3×12,4×9,6×6
√36 = 6 なので、6以降の因数（9, 12, 18）はすでに6以前の因数（4, 3, 2）とペアになっている。だから6まで調べれば十分。
